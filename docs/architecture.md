# GoAI Architecture

## **Overview**
GoAI is a modular AI framework built in **Go**, enabling **secure, scalable AI models** integrated with **blockchain**.

## **Components**
### **1. Core Engine**
- Handles model training & inference
- Uses matrix operations for efficiency

### **2. Blockchain Module**
- Secures AI models on-chain
- Provides model validation

### **3. CLI & API**
- Command-line tool (`goai-cli`)
- REST API for model management
